## 바로 구현 추천 순서

1. 골든크로스
2. 추세 필터
3. 평균 회귀
4. 이격도
5. 모멘텀

## 추천 이유

- 골든크로스: 현재 계산된 이동평균만으로 바로 구현 가능하고 이해가 쉽다.
- 추세 필터: 장기 이동평균 기반이라 과도한 매매를 줄이기 좋다.
- 평균 회귀: 기준선 대비 얼마나 벗어났는지 보기 쉬워 초보자에게 설명하기 좋다.
- 이격도: 평균 회귀와 비슷하지만 백분율 기준이라 임계값 설정이 명확하다.
- 모멘텀: 기간 수익률 기준으로 판단해 백테스트 확장이 쉽다.

## 보류 또는 주의가 필요한 전략

- 52주 신고가
  - 현재 reference는 현재가 API의 52주 고가 정보를 활용한다.
  - 우리 프로젝트에서는 아래 둘 중 하나가 필요하다.
  - 현재가 API 연동 추가
  - 또는 최소 250거래일 이상 일봉 데이터 확보 후 자체 계산

- 돌파 실패
  - 일봉으로 구현은 가능하지만, 단독 진입 전략보다 보조적인 청산 전략으로 쓰는 편이 더 자연스럽다.

- 강한 종가
  - 당일 고가/저가/종가 관계를 보는 전략이라 장 마감 이후 또는 마감 직전 데이터 기준 사용이 적합하다.

- 투자자 수급 기반 전략
  - 현재 reference 기본 10종에는 직접 포함되지 않지만, 우리 프로젝트에서는 차별화 포인트가 될 수 있다.
  - 예: 외국인 순매수 지속 + 종가가 20일선 위이면 매수 후보
