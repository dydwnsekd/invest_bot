# Indicator Guide

## 목적

이 문서는 `invest_bot`에서 현재 사용 중인 주요 지표가 무엇을 뜻하는지, 차트를 볼 때 어떻게 해석하면 되는지 정리한 가이드입니다.

현재 기준으로 설명하는 지표는 아래와 같습니다.

- `ma_5`
- `ma_20`
- `ma_60`
- `volume_ma_5`
- `rsi_14`

## 먼저 보면 좋은 해석 순서

차트를 처음 볼 때는 아래 순서로 해석하는 것이 가장 쉽습니다.

1. 현재 가격이 어느 이동평균선 위에 있는지 확인
2. `ma_5`와 `ma_20`의 위치 관계 확인
3. `rsi_14`로 과열/과매도 여부 확인
4. 거래량과 `volume_ma_5`를 비교해 움직임의 강도 확인

## 1. ma_5

### 의미

`ma_5`는 최근 5거래일 종가의 평균입니다.

### 무엇을 보여주나

- 아주 짧은 기간의 가격 흐름
- 최근 며칠 동안 가격이 빠르게 강해지는지, 약해지는지

### 해석 예시

- `close > ma_5`
  - 현재 가격이 최근 며칠 평균보다 강한 편일 수 있습니다.
- `close < ma_5`
  - 단기적으로 힘이 약해졌을 수 있습니다.

### 주의점

`ma_5`는 매우 민감해서 신호가 자주 바뀔 수 있습니다. 단독으로 보기보다 `ma_20`과 함께 보는 편이 좋습니다.

## 2. ma_20

### 의미

`ma_20`은 최근 20거래일 종가의 평균입니다.

### 무엇을 보여주나

- 약 한 달 안팎의 흐름
- 단기 추세가 우상향인지, 우하향인지 판단하는 기준선

### 해석 예시

- `close > ma_20`
  - 현재 가격이 한 달 평균보다 높아 비교적 강한 상태로 볼 수 있습니다.
- `close < ma_20`
  - 한 달 기준 흐름보다 약한 상태로 볼 수 있습니다.

### 왜 중요한가

`ma_20`은 `ma_5`보다 덜 흔들려서 단기 잡음을 줄이고 현재 방향을 읽기 좋습니다.

## 3. ma_60

### 의미

`ma_60`은 최근 60거래일 종가의 평균입니다.

### 무엇을 보여주나

- 더 긴 중기 흐름
- 큰 방향이 상승인지 하락인지 판단하는 기준

### 해석 예시

- `close > ma_60`
  - 중기적으로는 비교적 강한 흐름일 가능성이 있습니다.
- `close < ma_60`
  - 중기 기준으로는 아직 약한 흐름일 수 있습니다.

### 왜 중요한가

`ma_60`은 짧은 흔들림보다 큰 추세를 보는 데 유용합니다. `ma_20`보다 더 보수적인 기준선이라고 보면 됩니다.

## 4. volume_ma_5

### 의미

`volume_ma_5`는 최근 5거래일 평균 거래량입니다.

### 무엇을 보여주나

- 최근 평소 거래 수준
- 오늘 거래량이 평소보다 많은지 적은지 비교하는 기준

### 해석 예시

- `volume > volume_ma_5`
  - 오늘 움직임에 거래가 평소보다 더 붙고 있을 수 있습니다.
- `volume < volume_ma_5`
  - 가격이 움직여도 거래 참여는 크지 않을 수 있습니다.

### 왜 중요한가

가격 상승이나 하락이 나와도 거래량이 너무 약하면 신호 신뢰도가 낮을 수 있습니다.

## 5. rsi_14

### 의미

`rsi_14`는 최근 14기간 동안 가격 상승 강도를 수치화한 지표입니다.

### 값의 범위

- 일반적으로 `0 ~ 100`

### 자주 쓰는 해석 기준

- `70 이상`
  - 과열 가능성
- `30 이하`
  - 과매도 가능성
- `30 ~ 70`
  - 중립 구간으로 보는 경우가 많음

### 해석 예시

- `rsi_14 >= 70`
  - 최근 상승이 너무 빠르게 진행됐을 수 있습니다.
- `rsi_14 <= 30`
  - 최근 하락이 과도했을 수 있습니다.

### 주의점

RSI가 높다고 바로 하락하고, 낮다고 바로 상승하는 것은 아닙니다. 추세가 강한 장에서는 과열/과매도 상태가 꽤 오래 지속될 수 있습니다.

## 골든크로스와의 관계

현재 프로젝트는 `ma_5`와 `ma_20`을 이용해 골든크로스 전략을 사용합니다.

### 골든크로스

- 이전 시점: `ma_5 < ma_20`
- 현재 시점: `ma_5 > ma_20`

이 경우 보통 `buy` 신호 후보로 봅니다.

### 데드크로스

- 이전 시점: `ma_5 > ma_20`
- 현재 시점: `ma_5 < ma_20`

이 경우 보통 `sell` 신호 후보로 봅니다.

## 차트 해석 예시

### 예시 1. 비교적 강한 상승 흐름

- `close > ma_20`
- `ma_5 > ma_20`
- `rsi_14 = 58`
- `volume > volume_ma_5`

해석:
- 단기 추세는 상승 쪽
- 과열은 아직 심하지 않음
- 거래량도 붙고 있어 신호가 비교적 건강할 수 있음

### 예시 2. 과열 가능성 있는 상승 구간

- `close > ma_20`
- `ma_5 > ma_20`
- `rsi_14 = 76`

해석:
- 상승 추세는 유지되고 있을 수 있음
- 다만 과열 구간이라 바로 추격 매수는 조심할 필요가 있음

### 예시 3. 약한 흐름

- `close < ma_20`
- `ma_5 < ma_20`
- `rsi_14 = 42`

해석:
- 추세는 아직 약한 편
- 과매도까지는 아닐 수 있음
- 명확한 반등 신호가 나오기 전까지는 관망 해석이 더 자연스러울 수 있음
